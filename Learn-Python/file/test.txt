一直以来，美国按照其“美国利益优先”原则，在国际上实施霸凌主义，动辄以经济制裁、军事威胁、政治渗透、
外交围堵等手段，以所谓的“国家安全”为由，对别国实施打压，破坏国际秩序，引发矛盾冲突和安全危机。近期，
还通过对企业实施打压，展开了一场数字时代的“科技战争”。美国的作法，已经引起国际社会的强烈反对，已到
千夫所指的程度。

据环球网6月8日援引路透社等外媒报道称，在日前召开的第23届圣彼得堡国际经济论坛上，俄罗斯总统普京
公开批评美国，认为美国打压华为，是奉行“肆无忌惮的经济利己主义”，认为美国使用施压与恐吓的手段“进
行公开的经济掠夺”，普京甚至认为美国在打压华为上的做派，是“数字时代的第一场科技战争”。与此同时，
在这次圣彼得堡国际经济论坛上，俄罗斯最大的移动运营商MTS首席执行官阿列克谢·科尔尼也表示，华为“
完全有资格”为俄罗斯下一代无线电网络提供基础设施。

在美国不断对华为实施打压，甚至给其他国家施压的前提下，俄罗斯总统普京在国际论坛上力挺华为，公开
向美国叫板，认为美国打压华为包括“厚颜无耻”地想将华为赶出市场，完全是美国在搞鬼，普京总统的发言，
无疑是为中国企业受到美国的无端打压而两肋插刀；而俄罗斯的移动运营商对华为的盛赞，同样是对美国打压
华为作法的打脸。俄罗斯在华为事件上的态度，既是对美国经济霸权主义的强烈抨击，也为华为未来发展提供了支持。
