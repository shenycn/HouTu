### 内容与地址
1. 内存由很多内存单元组成，用于存放各种类型的数据。
2. 计算机对内存的每个内存单元都编了号（内存地址），地址决定了内存单元在内存中的位置
3. C++语言编译器通过名字来访问内存位置

### 关于变量、地址和指针变量
1/一个变量有三个重要信息
1. 变量的地址位置
2. 所存储的信息
3. 变量的类型（记录在编译器）

2/指针变量
专门用来记录变量地址的变量；通过指针变量可以间接访问另一个变量的值（有可能仍是一个指针变量）

| 地址 | `116` | 112  | `108` | 104        | 110   |
| ---- | :---- | ---- | ----- | ---------- | ----- |
| Data| 112  | -1    | 1078523331 | `116` |`108`|

```cpp
int a = 112, b = -1;
float c = 3.14f;
int *d = &a;
float *e = &c;
cout << d << endl;
cout << e << endl;
cout << *d << endl;
cout << *e << endl;
```
